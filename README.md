## 安装

```bash
npm i --save syswing-react
```

👍 or yarn

```bash
yarn add syswing-react
```

## 开发组件

* 构建

```bash
npm run build
```

* 开发

```bash
npm run start
```

* 使用nrm

```bash
npm install -g nrm  // 安装
nrm use taobao      // 切换taobao镜像
nrm use npm      // 切换npm镜像
```
